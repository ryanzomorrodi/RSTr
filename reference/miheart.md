# Michigan Heart Attack Mortality Data

A dataset containing counts for Myocardial Infarction (ICD-9 code 410.0)
deaths in all 83 Michigan counties for individuals in 6 age groups
across 10 years. This dataset also contains the corresponding population
counts.

## Usage

``` r
miheart
```

## Format

\`miheart\` a list with two array objects:

- Y:

  death count (0-1005)

- n:

  population count (26-292828)
